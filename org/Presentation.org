* Optimization - overview
** Autotuning optimization problem
- Nonsmooth objective function and constraints
- Discrete and continuous

** Knowledge about the problem
*** Know shape of the problem
- Exploit information about the problem
- Derivative based methods
- Use of analytic models \to surrogates-based search
*** No knowledge
- Optimization problem not differentiable \to cannot build reliable
  model
- Derivative-free based methods
- Based on heuristics \to Genetic algorithm, simplex, pattern search
*** Partial knowledge
- Derivative informations not available everywhere
** Search space strategy
*** Local:
- Focus on a particular area 
- Effective when knowing where to concentrate the search
- The optimal found can be far worst than the global optimum
- E.g: simplex, surrogates-based search
*** Global :
- Cover various part of the search space
- Effective when no knowledge about the problem
- Escape bad local optimum
- E.g: Random, Genetic algorithm

** Mixed strategy
*** Global and local search
- Two phases 
- Global \to escape bad local optima
- Local \to refine solution
- E.g pattern search, simulated annealing
*** Derivative and derivative-free
- Partial knowledge
- Reduce exploration time
- E.g.: Generalized Pattern Search 

** Generalized Pattern Search
- Extended pattern search version
- For unconstrained and linearly constrained problems
- Iteration over two phases:
  - Global search \to SEARCH
    - Sampling the space to find interesting regions \to building a mesh
    - Try to improve current optimal elsewhere
    - Possible to use any methods \to Genetic Algo, surrogates based
      search,  etc...
  - Local search \to POLL
    - Exploiting interesting region to refine the solution
- Uses derivative informations to speedup POLL phase
